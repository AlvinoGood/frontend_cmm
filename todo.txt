Implement auth interfaces and logic

* **Login (DNI + password):** store JWT + decode `sys_role`, `medical_role`. Token of 5 minutes.
* **Register:** DNI, email, password → default role `user`.
* **Forgot/Reset:** send reset token/link to the user’s email (email token flow; clean pastel screens, friendly messages).
* **Expired token:** refresh token. (logic of authentication)
